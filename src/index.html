Site API
  • Users
  • Groups

Please before sending a request, get a token through http://localhost:3000/login (POST)

Provide the token in heading x-access-token for every request.

  • view all users: http://localhost:3000/users
  • create user: http://localhost:3000/users (POST). Request example is in assets/create-user.json
  • view user: http://localhost:3000/users/1
  • update user: http://localhost:3000/users/1 (PUT)
  • delete user: http://localhost:300/users/1 (DELETE). Peforms a soft delete, `isDeleted` = true.
  • view all groups: http://localhost:3000/groups
  • create group: http://localhost:3000/groups (POST). Request example is in assets/create-group.json
  • view group: http://localhost:3000/groups/1
  • update group: http://localhost:3000/group/1 (PUT)
  • delete group: http://localhost:300/group/1 (DELETE)
  • add users to group: http://lolcahost:3000/groups/1 (POST). Request example in assets/add_users_to_group.json
